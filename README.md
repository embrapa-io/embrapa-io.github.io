## Development

To change and improve the site content, run [Jekyll](https://jekyllrb.com) locally:

```
docker run --rm --name jekyll -p 8123:4000 -v $(pwd):/site bretfisher/jekyll-serve:alpine
```

And access: http://localhost:8123
